A simple yet advanced Cookie Wordpress plugin with unlimited cookies, statistics and real time editing and much more.

== Description ==

The WordPress Cookie Plugin allows you to create unlimited cookies with visual presented statistics and real time editing.

The plugin displays a cookie notice placed on the website site by your choice.

Visit plugin website <https://www.jesmadsen.com/wordpress-plugins/wordpress-cookie-plugin/> for more information about the plugin.



== Installation ==
Download: [WordPress Cookie Plugin](https://wordpress.org/plugins/cookies-by-jm/ "WordPress Cookie Plugin")

1. Install the plugin via WordPress.org or unzip the contents of the file into your wp-content/plugins folder.

2. Activate the plugin

3. Create your first cookie

4. See statistics for cookie impressions and accepted cookies


== Features ==

Unlimited cookies
Realtime design preview in backend
Accepts option on either button or close icon
Display duration
Option to show the notification on the first page only
Customizable design and placement
100% Responsive
Visual statistics on impressions and accepted cookies
Button link to internal or external page
